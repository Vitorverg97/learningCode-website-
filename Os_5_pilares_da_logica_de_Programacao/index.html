Os 5 Pilares Básicos Para Dominar a Lógica De Programação

Tipos de Dados

Os tipos de dados definem que tipo de informação pode ser armazenada e manipulada em um programa. Compreender os diferentes tipos ajuda a escolher o mais adequado para cada situação.
Os principais tipos de dados:

  • Numéricos: Inteiros (INT); Ponto Flutuante (Float, Double), etc.
  • Textuais: Strings (Cadeia de caracteres).
  • Booleanos: Representam valores Lógicos (TRUE / FALSE).
  • Coleções: Listas, Arrays Dicionários (mapas), conjuntos (sets), etc.

Exemplo: 

 
idade = 25      # Inteiro
preco = 19,99   # float
nome = "joão"   # String
ativo = TRUE    # Booleano

Operadores

Os operadores são usados para realizar operações em variáveis e valores. Eles são essenciais para realizar cálculos, comparações e manipulações de dados.

Categorias de operadores:

  • Operadores Aritméticos: +, -, *, /, %(Módulo), **(Exponenciação). (Python)
  • De Comparação: ==, !=, <, >, <=, >=.
  • Lógicos: and, or, not.
  • De atribuição: =, +=, -=, etc.

Exemplos: 

 
a = 10
b = 20
soma = a + b → 30
comparacao = a > b → False

Estruturas de Controle

As estruturas de controle permitem tomar decisões no código e executar blocos de forma condicional ou repetitiva.

Tipos Principais:

Condicionais: if, else, elif (em Python). 

    
    if idade >= 18:
        print("Maior de idade")
    else:
        print("Menor de idade")

Laços de repetição (loops): for, while.

    
    for i in range(5):
        print(i)  # Imprime de 0 a 4

Estruturas de Dados

Estruturas de dados são formas de organizar e armazenar informações. Elas são fundamentais para resolver problemas complexos de forma eficiente.

Exemplos comuns:

Listas: Sequências ordenadas de elementos.


    frutas = ["maçã", "banana", "cereja"]

Dicionários: Coleção de pares chave-valor. 

aluno = {"nome": "João", "idade": 18}

Conjuntos: Conjuntos únicos e não ordenados. 

numeros = {1, 2, 3, 4}

Funções

As funções permitem encapsular código reutilizável, tornando-o mais organizado e modular. Conceitos importantes:

  • Definição: Bloco de código nomeado que pode receber entradas (parâmetros) e retornar um resultado.
  • Reutilização: Escreva uma vez e use várias vezes.
Exemplo: 

def soma(a, b):
    return a + b
    
resultado = soma(5, 3)  # Chama a função e armazena o retorno
print(resultado)        # 8
